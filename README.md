# Bit Brains Website

This repository contains the public-facing website for the Bit Brains protocol.

## Purpose

- Provide protocol overview and documentation access
- Surface Proof of Care and staking mechanics
- Act as the canonical interface for participants, stewards, and observers

## Architecture (Initial)

- Frontend-first (static + reactive)
- Protocol-aware, not protocol-custodial
- Wallet-connected read/write where applicable
